# Baskin-Robbins-Sales-Performance-Dashboard
## Project Overview
This project presents a Sales Performance Dashboard built using Power BI to analyze and visualize the business performance of a Baskin Robbins outlet network.
The dashboard provides insights into sales trends, order patterns, payment methods, category performance, and seasonal analysis, helping stakeholders make data-driven decisions.
## Business Objective
To analyze:

-Total Sales Performance
-Order Volume & Average Order Value

Sales by Category

Payment Mode Distribution

Yearly & Quarterly Sales Trends

Sales by Time of the Day
